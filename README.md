## Tech Stack
- Frontend: React.js / Express
- Backend: Node.js
- Database: MongoDB
- Hosting: Vercel
- Other Tools: Git, GitHub, Tailwind CSS, Postman

## How to Run the Project Locally

### Prerequisites
- Node.js and npm installed
- Git installed
- MongoDB Compass

---

### To run website
- git clone https://github.com/HemaSai777/EDULEARN
- npm install
- npm start
